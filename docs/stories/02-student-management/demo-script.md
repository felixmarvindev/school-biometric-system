# Story 02: Student Management - Demo Script

## Demo Overview

**Duration**: 4-5 minutes  
**Goal**: Show how admins register and organize students

## Demo Flow

### 1. Create Class Structure (1 minute)
- Show class creation form
- Create sample classes (Grade 1, Form 1)
- Create streams (A, B, C)
- Explain organization

### 2. Add Student (1.5 minutes)
- Click "Add Student"
- Fill registration form
- Show validation
- Submit and show success
- Student appears in list

### 3. Student List (1 minute)
- Show student list
- Demonstrate search
- Demonstrate filtering by class
- Show student count

### 4. Student Details (1 minute)
- Click student to view details
- Show complete profile
- Show parent information
- Demonstrate edit capability

## Key Talking Points

- "Students are the foundation of the attendance system"
- "Classes and streams organize students by grade level"
- "Parent contacts enable SMS notifications"
- "All student data is securely stored and organized"

---

**Focus on ease of use and organization capabilities.**
