# Story 03: Device Management - Acceptance Criteria

## Story-Level Acceptance Criteria

- [ ] Devices can be registered with IP, port, serial number
- [ ] Device connection status is tracked and displayed
- [ ] Device groups can be created and managed
- [ ] Devices can be assigned to groups
- [ ] Real-time status updates work via WebSocket
- [ ] Simulation mode enables demos without physical devices
- [ ] Device capacity and enrollment count tracked
- [ ] Connection testing works correctly

---

**All criteria must be met before marking story complete.**
