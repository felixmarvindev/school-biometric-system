# Story 05: Attendance Tracking - Demo Script

## Demo Overview

**Duration**: 4-5 minutes  
**Goal**: Show real-time attendance tracking capabilities

## Demo Flow

### 1. Real-time Dashboard (2 minutes)
- Show attendance dashboard
- Simulate student check-in (or use device)
- Show event appearing instantly
- Show IN/OUT indicators
- Show multiple events

### 2. Entry/Exit Logic (1 minute)
- Explain automatic IN/OUT determination
- Show first tap is IN
- Show second tap is OUT
- Show duplicate prevention

### 3. Attendance History (1 minute)
- Show history view
- Demonstrate filtering
- Show individual student history

## Key Talking Points

- "Attendance events appear instantly"
- "System automatically determines IN/OUT"
- "Complete history available for reporting"

---

**Emphasize real-time capabilities and automatic entry/exit determination.**
