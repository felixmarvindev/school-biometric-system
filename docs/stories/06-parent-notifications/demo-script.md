# Story 06: Parent Notifications - Demo Script

## Demo Overview

**Duration**: 3-4 minutes  
**Goal**: Show automatic SMS notifications to parents

## Demo Flow

### 1. Trigger Notification (1.5 minutes)
- Show attendance event (check-in)
- Explain notification is triggered
- Show SMS being sent
- Show phone receiving message

### 2. Show Message Content (1 minute)
- Show actual SMS content
- Explain message format
- Show both IN and OUT messages

### 3. Delivery Tracking (1 minute)
- Show notification status
- Explain delivery tracking
- Show retry logic (if applicable)

## Key Talking Points

- "Parents receive instant notifications"
- "SMS sent automatically on every check-in/out"
- "Delivery is tracked and failures are retried"

---

**Emphasize the value to parents and peace of mind.**
