# Story 07: Reporting & Analytics - Demo Script

## Demo Overview

**Duration**: 4-5 minutes  
**Goal**: Show reporting and analytics capabilities

## Demo Flow

### 1. Daily Report (1.5 minutes)
- Show daily report page
- Display statistics
- Show present/absent lists
- Demonstrate date filtering

### 2. Student History (1.5 minutes)
- Show student attendance history
- Display attendance rate
- Show timeline
- Explain patterns

### 3. Class Summaries (1 minute)
- Show class summary page
- Display class attendance rates
- Show comparisons
- Explain insights

## Key Talking Points

- "Comprehensive reporting for data-driven decisions"
- "Easy to identify attendance issues"
- "Reports can be exported for stakeholders"

---

**Emphasize the value of insights and data-driven decisions.**
